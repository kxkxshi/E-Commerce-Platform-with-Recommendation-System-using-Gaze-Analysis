# 🛒 E-Commerce Platform with Gaze-Based Recommendation System

## 🚀 Overview

This project is an advanced e-commerce platform that enhances user experience using **gaze-based recommendation systems**.
Instead of relying only on clicks and purchases, the system tracks user attention (eye gaze) to understand interests in real-time and provide personalized product recommendations.

---

## 🧠 Key Idea

Traditional recommendation systems rely on:

* Clicks 
* Purchase history

This project introduces:
👉 **Gaze Tracking → Attention Data → Smarter Recommendations**

---

## ⚙️ How It Works

1. User browses products
2. Eye gaze is tracked using WebGazer.js
3. Gaze points are converted into attention heatmaps
4. Products with higher attention are prioritized
5. Recommendation engine suggests relevant items

---

## 🏗️ Architecture

User → Frontend UI → Gaze Tracking (WebGazer.js)
→ Data Processing → Recommendation Engine → Product Suggestions
